    Examples

    Before diving into creating your own XRPackages, it can be useful to see some examples and play around with the code!

    If you think something is missing from this list, you can submit a pull request on GitHub.

    GLB XRPKs#

    • Parkour - A big industrial land scene.
    • Home - A home environment scene.
    • Kokiri Forest - A Kokiri Forest scene.
    • Glorb - A wearable model.
    • Tip Jar - A tip jar model.

    ThreeJS Website XRPKs#

    • Comet - A comet.
    • Land - A colorful geometric land.
    • Planet - A big geometric planet.
    • Voxels - A collection of voxels.
    • Physics Cube - A cube with physics.
    • Portal - A set of portals.
    • Lightsaber - A lightsaber.
    • Mirror - A mirror.
    • Sheild - A shield.
    • Weapons - A table of weapons.
