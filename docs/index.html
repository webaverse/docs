â“ What is the Webaverse?

The Webaverse is a blanket term used to describe a set of technologies that enable the use of spatial applications and objects powered by WebXR. These spatial applications are called "XR Packages" (XRPKs).

See What Is Webaverse? for a more in-depth explanation, or read on to get started with using and developing XRPKs!

See Getting Started for detailed guides on how to get started, as a user or developer!

â“ What Are XR Packages (XRPKs)?

XR Packages (XRPKs) are creative spatial apps that run everywhere. Unlike traditional 2D applications, XRPKs exist in three dimensions and can be experienced with a desktop browser or any SteamVR compatible headset. XRPKs are created and owned by users like you! Getting started with XRPKs today is easy, keep reading below to get started, or see the more page What Is Webaverse? for a more in-depth explanation.
